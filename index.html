CSS Button Cheat Sheet

CSS Selector – button
Property Description Example
color Sets the color of the text or foreground color: white;
background-color Sets the color of the text or background background-color: darkolivegreen;
font-size Sets the size of the text font-size: 18px;
font-family Sets the font family of the text font-family: Calibri, Sans;
margin Sets the space from other elements margin: 20px auto;
padding Sets the space from inner elements padding: 15px;
border Sets the border style border: none;
border-radius Sets the rounding of the border corners border-radius: 10px;
box-shadow Sets the shadow of the element box-shadow: 2px 2px 0 0 darkgray;
transition Sets the transition for style changes transition: all 0.2s;

CSS Selector – button:hover
Property Description Example
cursor Sets the style of the cursor when hovered cursor: pointer;
background-color Sets the backround color on hover background-color: olivedrab;

CSS Selector – button:active
Property Description Example
background-color Sets the backround color when clicked background-color: olive;
